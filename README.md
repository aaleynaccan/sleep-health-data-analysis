# Sleep Pattern & Health Indicator Analysis
Business Intelligence project analyzing sleep patterns and health indicators using Power BI and Excel.

## Project Overview
This project analyzes sleep behavior and health indicators in a working population to identify factors that influence sleep quality and duration.

The analysis focuses on relationships between sleep duration, stress levels, physical activity, BMI, and other health indicators.

## Dataset
The dataset was obtained from Kaggle and contains 374 observations with the following variables:

- Sleep Duration
- Quality of Sleep
- Stress Level
- Physical Activity Level
- BMI Category
- Blood Pressure
- Heart Rate
- Daily Steps
- Occupation
- Sleep Disorders

Source: Kaggle Public Dataset

## Tools Used
- Power BI
- Microsoft Excel

## Analytical Methods
The project applied several statistical and exploratory data analysis techniques:

- Univariate analysis
- Bivariate correlation analysis
- Multivariate analysis
- Data visualization
- Dashboard development

## Key Insights
The analysis revealed that stress level has the strongest negative relationship with sleep duration and sleep quality.

Physical activity demonstrated a weaker but positive influence on sleep health. These findings suggest that workplace wellness initiatives should prioritize stress management strategies while also promoting regular physical activity.

## Deliverables
Interactive dashboards were developed in Power BI to visualize patterns and relationships between health indicators and sleep behavior.

The dashboards include:
- Sleep duration distribution
- Stress vs sleep analysis
- BMI category breakdown
- Activity and sleep relationship visualizations
